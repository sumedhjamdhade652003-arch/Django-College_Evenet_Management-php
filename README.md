# Django-College_Evenet_Management-php
PROJECT INTRODUCTION
The College Events Management Portal is a full-stack web application designed to centralize and automate campus event coordination. It bridges the gap between students and organizers by replacing manual, scattered scheduling with a seamless digital ecosystem.

The platform delivers a dual-experience: an interactive, responsive user portal where students can browse and register for Technical, Cultural, and Sports events, alongside a secure administrative dashboard that allows coordinators to manage event details, deadlines, and prize structures in real time.

CORE TECH STACK
Frontend: HTML, CSS, Bootstrap, Tailwind CSS

User Backend: PHP

Admin Dashboard: Django

Database: MySQL
